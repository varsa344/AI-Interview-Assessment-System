# AI Interview Assessment System

An AI-powered interview assessment platform that evaluates spoken interview responses using speech intelligence, NLP, and Large Language Models.

## Features

* Speech-to-Text using Whisper
* Filler Word Detection
* Fluency Analysis
* Confidence Estimation
* Grammar Checking
* Sentiment Analysis
* AI-Generated HR Feedback
* Interview Scoring

## Tech Stack

* Streamlit
* Whisper
* Librosa
* LanguageTool
* Transformers
* Groq LLM

## Workflow

Audio Input → Speech Recognition → Speech Analytics → NLP Evaluation → Scoring → AI Feedback
